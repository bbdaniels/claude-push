Claude Push

Get Apple Watch haptic notifications when Claude Code needs your attention. Uses ntfy.sh -- no account, no API key, no webhook URL.

No Account Needed

Built on ntfy.sh -- just pick a topic name and go. No API keys, no webhook URLs, no signup required.

PreToolUse Hooks

Fires on AskUserQuestion events so you get a tap on your wrist whenever Claude needs input.

VSCode Workaround

Works around the VSCode hook bug (#8985) by instructing Claude to use AskUserQuestion proactively.

Instant Setup

Install ntfy on your phone, add a JSON block to settings, restart Claude Code. Done in under a minute.

How It Works

Claude asks question (PreToolUse hook)
    -> curl POST to ntfy.sh
    -> ntfy sends push notification
    -> iPhone/Apple Watch receives haptic ping

Quick Start

1. Install ntfy

Download ntfy from the iOS App Store and subscribe to a topic (e.g. claude-yourname-a1b2c3).

Privacy: ntfy topics are public. Anyone who knows the name can send to it. Use a random suffix to keep yours private.

2. Test the notification

curl -d "test" ntfy.sh/YOUR_TOPIC

3. Configure Claude Code

Add to ~/.claude/settings.json:

{
  "hooks": {
    "PreToolUse": [
      {
        "matcher": "AskUserQuestion",
        "hooks": [
          {
            "type": "command",
            "command": "curl -s -d 'Claude needs your attention' ntfy.sh/YOUR_TOPIC"
          }
        ]
      }
    ]
  }
}

4. Restart Claude Code

Hooks take effect after restart.

Hook Compatibility

Hook CLI VSCode
PreToolUse (AskUserQuestion) Yes Yes
PostToolUse Yes Yes
Notification Yes No
PermissionRequest Yes No

VSCode Workaround

Important: The PermissionRequest and Notification hooks don't work in VSCode due to a known bug (#8985).

Add this to your ~/.claude/CLAUDE.md to instruct Claude to proactively ask questions, which triggers the working PreToolUse hook:

## Push Notifications (VSCode)

When working in VSCode, use `AskUserQuestion` to confirm before taking
significant actions that would normally require permission. This triggers
a push notification to alert the user on their Apple Watch.
